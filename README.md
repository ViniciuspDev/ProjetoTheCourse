# 🎓 Projeto The Course

Site para divulgação do curso de inglês The Course Angra, desenvolvida com PHP puro, HTML, CSS e JavaScript.

---

## 🚀 Sobre o projeto

O Site foi feito do zero com auxílio de Inteligência Artificial para gerar imagens e textos genéricos.

O projeto foi desenvolvido com foco em aprendizado de lógica, design, estrutura web e conceitos fundamentais de desenvolvimento backend.

---

## 🛠️ Tecnologias utilizadas

* PHP
* HTML5
* CSS3
* JavaScript

---

## 📌 Funcionalidades

* ✅ Envio de E-mails
* ✅ Validação de formulários
* ✅ Política de privacidade com consentimento (LGPD)
* ✅ Interface web simples e funcional

---

## 🔐 Segurança

O projeto conta com medidas básicas de segurança:

* Validação de formulários (frontend e backend)
* RECaptcha
* Honeypot
* Consentimento explícito do usuário (LGPD)

> ⚠️ Este projeto ainda não está pronto para produção e pode ser aprimorado com autenticação mais robusta e banco de dados.

## ▶️ Como executar o projeto

1. Clone o repositório:

```bash
git clone https://github.com/ViniciuspDev/ProjetoTheCourse.git
```

2. Coloque a pasta em um servidor local:

* XAMPP
* WAMP
* Laragon

3. Inicie o servidor Apache

4. Acesse no navegador:

```
http://localhost/ProjetoTheCourse
```
